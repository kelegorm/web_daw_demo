**useTransportController.ts changes:**

1. Change the `useTransportController` function signature: replace `trackStrip: TrackStripHook` (2nd parameter) with `setTrackMuted: (muted: boolean) => void`.

2. Remove the `import type { TrackStripHook } from './useTrackStrip'` import.

3. Inside the function body, replace:
   ```
   const trackStripRef = useRef(trackStrip);
   trackStripRef.current = trackStrip;
   ```
   with:
   ```
   const setTrackMutedRef = useRef(setTrackMuted);
   setTrackMutedRef.current = setTrackMuted;
   ```

4. In the `createTransportCore` deps object (the `coreRef.current` initialization), change:
   ```
   setTrackMuted: (muted) => trackStripRef.current.setTrackMuted(muted),
   ```
   to:
   ```
   setTrackMuted: (muted) => setTrackMutedRef.current(muted),
   ```

5. The `createTransportCore` factory function and `TransportCoreDeps` interface are UNCHANGED — they already accept `setTrackMuted: (muted: boolean) => void`. Only the React hook wrapper changes.

6. Keep `isTrackMuted` in `TransportControllerState` — it is still needed by `onTrackMuteSync` in Layout.tsx and by Playwright tests.

**useSequencer.ts changes:**

1. Add `dispose(): void` to the `Sequencer` interface (after `setLoop`).

2. Add a `dispose` function inside `createSequencer` factory:
   ```typescript
   function dispose() {
     _active = false;
     _isPlaying = false;
     _currentStep = -1;
     _partStarted = false;
     part.stop(0);
     part.cancel(0);
     // Deliberately does NOT call transport.stop() — only the user-facing stop should halt global transport
     // Deliberately does NOT call panic() — caller decides whether to send note-offs
   }
   ```

3. Add `dispose` to the return object of `createSequencer`.

**Why NOT change createTransportCore:** The factory already accepts `setTrackMuted` as a plain callback in `TransportCoreDeps`. The coupling exists only in the React hook wrapper where `TrackStripHook` is destructured. The existing tests for `createTransportCore` already pass `setTrackMuted` as a plain mock — no test changes needed.
