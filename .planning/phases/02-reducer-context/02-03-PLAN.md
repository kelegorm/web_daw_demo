Create `src/context/` directory and the following files:

**src/context/DawProvider.tsx:**

Define three contexts as module-level constants (not exported — only accessible via hooks):
```
const ProjectContext = createContext<ProjectDocument | null>(null)
const UiContext = createContext<UiState | null>(null)
const DispatchContext = createContext<DawDispatch | null>(null)
```

Define `DawDispatch` interface (exported):
```typescript
export interface DawDispatch {
  addTrack(): string
  removeTrack(id: string): void
  selectTrack(id: string): void
}
```

`DawProvider` component:
- Props: `{ store: DawStore; children: React.ReactNode }`
- Uses `useSyncExternalStore(store.subscribe, store.getProjectSnapshot)` for project state.
- Uses `useSyncExternalStore(store.subscribe, store.getUiSnapshot)` for ui state.
- Creates dispatch object via `useMemo(() => ({ addTrack: () => store.addTrack(), removeTrack: (id) => store.removeTrack(id), selectTrack: (id) => store.selectTrack(id) }), [store])`.
- Renders nested contexts using React 19 syntax (NOT `.Provider`):
  ```jsx
  <ProjectContext value={project}>
    <UiContext value={ui}>
      <DispatchContext value={dispatch}>
        {children}
      </DispatchContext>
    </UiContext>
  </ProjectContext>
  ```

Export `ProjectContext`, `UiContext`, and `DispatchContext` as named exports (needed by hooks in separate files).

**src/context/useProjectState.ts:**
- `export function useProjectState(): ProjectDocument`
- Calls `useContext(ProjectContext)`.
- Throws descriptive error if context is null: `"useProjectState must be used within DawProvider"`.

**src/context/useUiState.ts:**
- `export function useUiState(): UiState`
- Calls `useContext(UiContext)`.
- Throws descriptive error if context is null: `"useUiState must be used within DawProvider"`.

**src/context/useDawDispatch.ts:**
- `export function useDawDispatch(): DawDispatch`
- Calls `useContext(DispatchContext)`.
- Throws descriptive error if context is null: `"useDawDispatch must be used within DawProvider"`.

**CRITICAL constraints:**
- Use React 19 `<Context value=...>` syntax, NOT `<Context.Provider value=...>` (deprecated in React 19).
- `useSyncExternalStore` (imported from `react`) — NOT a custom useEffect/useState subscription.
- No new runtime libraries — only React built-ins (STATE-08).
- DawStore's arrow-method class fields (subscribe, getProjectSnapshot, getUiSnapshot) are passed directly to useSyncExternalStore — no useCallback wrapping needed.
